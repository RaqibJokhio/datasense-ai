# DataSense AI

AI-powered data analyst — upload CSV/Excel files, ask questions in plain English, and get pandas-generated insights, auto-charts, and anomaly detection powered by an LLM.

## Features
- CSV/Excel upload
- Natural language querying of your data
- LLM-generated pandas code, executed safely
- Auto-generated charts
- Anomaly detection

## Tech Stack
- Backend: FastAPI, Pandas
- Frontend: React
- LLM: OpenRouter API
